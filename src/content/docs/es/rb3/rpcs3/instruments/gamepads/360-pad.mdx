* Mandos de Xbox 360 inalámbricos **requieren un receptor USB Inalámbrico de Xbox 360 para Windows para conectarse a tu computadora.**   
![Receptor USB inalámbrico Gamepad PC adaptador para Xbox 360](/milohaxguides/assets/instruments/adapters/360.png "Receptor USB inalámbrico Gamepad PC adaptador para Xbox 360")
	¿Necesitas uno?

	* <a href="https://rolllimitless.com/xbox-360-wireless-receiver/" target="_blank" rel="noopener noreferrer">[Compra uno de Roll Limitless]</a>
	* <a href="https://www.amazon.com/dp/B076GZFLR3" target="_blank" rel="noopener noreferrer">[Compra uno de Amazon]</a>
	* ¡Mayoría de los Receptores USB Inalámbrico Gamepad PC Adaptador Para Xbox 360 funcionan!
    * El kit de “Play and Charge” (Cargar y Jugar) solo funciona para cargar, no para conectar.
* No existe un perfil porque la configuración es por defecto.
* Este control tiene auto-strum (auto-rasguea).
* Necesitas Rock Band 3 Deluxe si quieres tocar Guitarra/Bajo/Teclado/Batería con mando (Padtar).
	- También puedes cambiar el mapeo dentro del juego por Deluxe.  
	`Menú > Opciones > Configuración Deluxe > Avanzado > Controlador > Re-Mapeo de Control`
