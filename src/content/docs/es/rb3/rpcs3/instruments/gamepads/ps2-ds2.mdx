* No existe un perfil para este mando porque varia mucho por adaptador.
* Esta información solo fue verificada con el adaptador "OSTENT PS1 PS2 To PC USB 2.0 Control Adapter Converter For Sony PS2 Wired Control" (Numero de modelo/SKU: 014800). Otros adaptadores pueden tener problemas.
* Es recomendado comprar otro adaptador.
	* Compra uno <a href="https://shop.retrocultmods.com/products/ps2-adapter-for-guitar-drum-dancepad-and-gamepad-controllers" target="_blank" rel="noopener noreferrer">[de RetroCultMods]</a>
		* <a href="https://www.etsy.com/listing/4309870670" target="_blank" rel="noopener noreferrer">[Enlace de Etsy]</a>
	* Compra el <a href="https://www.etsy.com/listing/1397089947" target="_blank" rel="noopener noreferrer">[Blue Indigo]</a>.
* Este control tiene auto-strum (auto-rasguea).
* Necesitas Rock Band 3 Deluxe si quieres tocar Guitarra/Bajo/Teclado/Batería con mando (Padtar).
	- También puedes cambiar el mapeo dentro del juego por Deluxe.  
	`Menú > Opciones > Configuración Deluxe > Avanzado > Controlador > Re-Mapeo de Control`
