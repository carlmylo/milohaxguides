    * Recuerda configurar cuales platillos tienes conectados dentro de Rock Band 3.
        * `Menu > Opciones > Opciones de Batería`

### Configuración inicial
Primero, vamos a descargar la versión mas actual de <a href="https://www.python.org/downloads/" target="_blank" rel="noopener noreferrer">[Python 3]</a>.  
<a href="https://www.python.org/downloads/" target="_blank" rel="noopener noreferrer">[Haz click aquí para ir a la pagina de Python 3]</a>.  
<a href="https://www.python.org/downloads" target="_blank" rel="noopener noreferrer"><Image src={pydl} alt="La pagina de descarga para Python. El cursor esta sobre 'Download Python 3.12.6.'" /></a>

Cuando termine, abre el instalador.  
En el instalador:  
1. Activa <span title="Agregar python.exe a PATH (ruta)">"<code>Add python.exe to PATH</code>"</span>
2. Click <span title="Instalar ahora">"<code>Install now</code>"</span>  
<Image src={pyinstall} alt="El instalador de Python. 'Add Python.exe to PATH' ha sido resaltado e activado y el cursor esta sobre 'Install now.'" />

Ahora, ve al <a href="https://github.com/yanagiragi/Minatsuki" target="_blank" rel="noopener noreferrer">[Github de Minatsuki]</a> y descarga el ZIP que contiene el código.  
<a href="https://github.com/yanagiragi/Minatsuki/archive/refs/heads/main.zip" target="_blank" rel="noopener noreferrer"><Image src={mina1} alt="La pagina de Minatsuki. El botón de 'Code' fue presionado y el cursor esta sobre la opción de 'Download ZIP.'" /></a>
Extrae el archivo `.zip` a una carpeta.  

Después de eso, ve al <a href="https://github.com/stefman69/Minatsuki-Pro-Drums-Plus" target="_blank" rel="noopener noreferrer">[Github de Minatsuki-Pro-Drums-Plus]</a> y descarga el ZIP que contiene el código.  
<a href="https://github.com/stefman69/Minatsuki-Pro-Drums-Plus/archive/refs/heads/main.zip" target="_blank" rel="noopener noreferrer"><Image src={mina2} alt="La pagina de Minatsuki-Pro-Drums-Plus'. El botón de 'Code' fue presionado y el cursor esta sobre la opción de 'Download ZIP.'" /></a>

Combina los archivos de Minatsuki-Pro-Drums-Plus con los archivos de Minatsuki.  
Después de eso, haz click arriba en la barra de direcciones del Explorador de archivos y escribe `cmd`. Presiona Enter.  
<Image src={mina3} alt="Una carpeta que tiene los archivos de Minatsuki y Minatsuki-Pro-Drums-Plus combinados. La barra de direcciones dice 'cmd'." />


En el intérprete de comandos, escribe `pip install -r requirements.txt` y presiona Enter.
<Image src={mina4} alt="El intérprete de comandos de Windows. El comando de 'pip install -r requirements.txt' fue escrito y se descargaron los requisitos." />

Después de que se instale eso, escribe `pip install vgamepad` y presiona Enter.  
<Image src={mina5} alt="El intérprete de comandos de Windows. El comando de 'pip install vgamepad' fue escrito y se descargo" />

Puede ser que también necesitas instalar ViGEm Bus. Puedes descargar esto de <a href="https://vigembusdriver.com/download/" target="_blank" rel="noopener noreferrer">[aquí]</a>.

### Uso
Sincroniza tu batería al receptor de Bluetooth de tu computadora.
<Image src={ps41} alt="El menu de Windows para dispositivos de Bluetooth. Muestra una batería de Mad Catz para Rock Band 4." />

Haz doble click en `main_xinput.py`.  
<Image src={ps42} alt="La carpeta que tiene los archivos de Minatsuki y Minatsuki-Pro-Drums-Plus combinados. El cursor esta sobre el archivo de 'main_xinput.py' y esta elegido" />

Cuando se abra el intérprete de comandos, escribe el numero de tu batería y presiona Enter. Aveces, la batería sale varias veces pero no hay problema. Solo elige una y presiona Enter.  
<Image src={ps43} alt="El intérprete de comandos de Windows con la script de 'main_xinput.py' abierta. Muestra la batería de Rock Band 4 y otros controles. Un numero de la batería esta escrito y Enter fue presionado." />

Minimiza este programa. **Asegúrate que se quede abierto mientras estas jugando**.

## Perfil

[[Descargar perfil]](https://github.com/hmxmilohax/rb3-pc/raw/refs/heads/main/downloads/instrument-repo/PS4%20Rock%20Band%20Drums.7z)
<Aside type="caution">Esto no se recomienda si estas conectando mas de un instrumento a la vez.</Aside>

Después de descargar el perfil,

1. Extrae el archivo .7z.
2. Arrastra la carpeta `input_configs` a la carpeta de `configs` en la carpeta donde tienes RPCS3.  
><Image src={instrepoinstall} alt="Instalando un perfil de la pagina de instrumentos." />

3. Después de eso, puedes elegir el perfil en el menú de `Mandos`.  
><Image src={rpcs3padprofile} alt="Gamepad Settings dentro de RPCS3 con el cursor sobre un perfil." />

La mayoría del tiempo, estos perfiles funcionan sin necesitar configuración adicional. Por si acaso no, cambia el control en `Dispositivos`, al lado del botón de `Actualizar` hasta que funcione. Puedes cambiar esto mientras el juego esté abierto.
