La mayoría de las canciones custom para Rock Band 3 están hechas para el Xbox 360. Igualmente, varios archivos oficiales (hechos por Harmonix) están en este formato. Existen como archivos CON que se pueden convertir para la PS3 e incluso el emulador RPCS3. Esta guía te va a enseñar como convertir estos archivos a formatos que puedes instalar fácilmente.

La guía va cubrir varios pasos para evitar problemas antes de que pasen. También hay una pagina para reparar problemas comunes como:
* Carga infinita/atorada (causado por encriptación incorrecta)
* Crasheos tocando o previsualizando canciones (causadas por calidad)
* Incompatibilidad jugando en linea y clasificaciones (causado por IDs incorrectos)

### Requisitos:
* Una consola PS3 con HEN o CFW (con una manera de meterle archivos) o <a href="https://rb3pc.milohax.org/gs_disc" target="_blank" rel="noopener noreferrer">[el emulador RPCS3]</a>
* <a href="https://rb3dx.milohax.org" target="_blank" rel="noopener noreferrer">[Una instalación de Rock Band 3 Deluxe]</a> (Requerido para ciertas partes)
* Una versión de <a href="https://nemosnautilus.com/nautilus" target="_blank" rel="noopener noreferrer">[Nautilus]</a> (antes conocido como C3 CON Tools)
	* <a href="https://github.com/trojannemo/Nautilus/releases" target="_blank" rel="noopener noreferrer">[Enlace alternativo]</a>
* <a href="https://github.com/mtolly/onyx/releases" target="_blank" rel="noopener noreferrer">[Herramientas de Onyx]</a> actuales.
