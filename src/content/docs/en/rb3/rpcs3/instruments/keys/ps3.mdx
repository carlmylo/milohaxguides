## Instructions
There are two different ways to connect this keyboard. If you don't have the Rock Band Keyboard Receiver for PlayStation 3, you will need to use MIDI to USB.  
Pick whichever is the most convenient for you:

<Tabs>
  <TabItem label="RB3 Keyboard Receiver for PlayStation 3 (Recommended)">
This is the recommended method because the keyboard will work as expected.

**You will need:**

* a PlayStation 3 Wireless Keyboard Receiver to connect to your computer.  
> ![PlayStation 3 Wireless Receiver](/milohaxguides/assets/instruments/buttons/ps3/reckeys.png "PlayStation 3 Wireless Keyboard Receiver")  

## Setup
<Steps>
1. Connect your Rock Band Keyboard Receiver for PlayStation 3 to your computer.

2. Now, **<a href="https://zadig.akeo.ie/" target="_blank" rel="noopener noreferrer">[go to Zadig's website]</a>** and **download the latest version, then open it** up.

3. Click on `Options` then `List All Devices`.  
    <Image src={zadiglistall} alt="Zadig showing 'List All Devices' under 'Options' highlighted." />

4. You should now see devices listed. **Switch it to your Rock Band 3 Wireless Keyboard Receiver**.  
In this example, it's the Pro Guitar (`Harmonix RB3 Mustang Guitar for PlayStationÂ® 3`).  
    <Image src={zadigsel} alt="Zadig showing 'Harmonix RB3 Mustang Guitar for PlayStationÂ® 3' highlighted in the devices listed." />

5. **After selecting the right device, you should see the option to replace the driver.  
_MAKE SURE YOU ARE REPLACING THE DRIVER ONLY FOR THE RB3 WIRELESS KEYBOARD RECEIVER!!_**  
Click `Replace Driver`.  
    <Image src={zadigreplace} alt="Zadig with 'Replace Driver' highlighted." />

6. A warning will appear. **Again, make sure you have selected your RB3 Wireless Keyboard receiver.** After you have made sure, **click** **`Yes`**.  
    <Image src={zadigwarning} alt="Zadig warning the user that they're about to modify a system driver, with 'Yes' highlighted" />

7. It will then install the driver. As the program says, it may take a few minutes.  
    <Image src={zadigprogress} alt="Zadig in the middle of a driver install." />

8. If everything goes well, you will get this message:  
    <Image src={zadigdone} alt="Zadig telling the user that the driver was installed successfully with 'Close' highlighted." />

9. **Close Zadig** and, **with the dongle** still **connected**, **open up RPCS3** and **open Rock Band 3**.
Turn your controller on and you should see it automatically assign a player number.  
    <Image src={protaron} alt="A Mustang Pro Guitar with the second player LED lit up." />
</Steps>

You will see the instrument ready to join in Rock Band 3, .  
<Image src={rb3player} alt="Rock Band 3 with a Pro Guitar ready to join." />


From this point, you can just connect your Rock Band 3 Wireless Keyboard receiver whenever you want without having to repeat this setup process!

<details>
<summary>Uninstalling</summary>
Applying a Zadig driver on a device will make it only work on applications with passthrough. Thankfully, reverting this is very easy.  
<Steps>
1. Open the Start menu and search for `Device Manager`. Open it.  
    <Image src={startdevman} alt="Windows' Start menu, with a result of 'Device Manager'." />

2. Scroll down until you find the `Universal Serial Bus devices` category.  
Expand it and right click the instrument you applied the Zadig driver on and then select `Uninstall device`.  
In this example, it's the Pro Guitar (`Harmonix RB3 Mustang Guitar for PlayStationÂ® 3`).  
    <Image src={devman} alt="Device Manager, with a right click menu that popped up. The cursor is over 'Uninstall Device.'" />

3. In the next window that pops up, make sure to tick `Attempt to remove the driver for this device.`  
4. After that, click `Uninstall` and that's it!  
    <Image src={devmanun} alt="The device uninstall prompt. 'Attempt to remove the driver for this device' is ticked and the cursor is over 'Uninstall.'" />
</Steps>
When you disconnect then reconnect the device, it will revert back to the original driver.

<Aside>**This requires advanced setup!**</Aside>

You can use your regular computer keyboard and convert its key presses to the MIDI notes that correspond to the game buttons in case you want to avoid swapping octaves constantly.

First, <a href="https://www.tobias-erichsen.de/software/loopmidi.html" target="_blank" rel="noopener noreferrer">[download loopMIDI]</a>.  
> <a href="https://www.tobias-erichsen.de/software/loopmidi.html" target="_blank" rel="noopener noreferrer"><Image src={midictrlloopMIDIdl} alt="loopMIDI's download page. The cursor is over 'download loopMIDI.'" /></a>

Install loopMIDI.  
**Launch it after it finishes.**  
> <Image src={midictrlloopMIDIinst} alt="loopMIDI's installer." />

> <Image src={midictrlloopMIDIaddport} alt="loopMIDI with a mouse cursor over the Plus symbol for 'Add Port'. Additionally, 'New port name' is highlighted in blue with a dotted outline, with 'Pro Keys' typed out in the text field." />

Now, download <a href="https://freepiano.tiwb.com/en/" target="_blank" rel="noopener noreferrer">[the `.zip` archive that contains the win64 version of FreePiano]</a>.  
> <a href="https://freepiano.tiwb.com/en" target="_blank" rel="noopener noreferrer"><Image src={midictrlfreepnodl} alt="FreePiano's download page'" /></a>

Extract the `.zip` archive somewhere you can easily find it.  
It was extracted to `C:\Games\freepiano` in this example.

Go to where you extracted FreePiano and run the `freepiano` executable.  
> <Image src={midictrlfreepnodir} alt="The FreePiano executable highlighted in a folder." />

Assign the keys to your liking as shown in the [**[#Additional Information]**](#additional-information) section above.

> <Image src={midictrlfreepnopreset} alt="A highlighted profile named rb3gamekeys.map in the keymap folder." />
