Ready to sing? RPCS3 lets you set up to 3 microphones to sing with.  
Just about any microphone can be used to sing.

To get started setting up microphones, make sure Rock Band 3 isn't running.  
**<a href="https://rb3pc.milohax.org/gs_quick_config" target="_blank" rel="noopener noreferrer">[Also, make sure you have installed a Quick Configuration Profile from the previous page]</a>**.  

<Steps>
1. **Right click on Rock Band 3** in RPCS3, then click “**`Change Custom Configuration`**”.
   <div>
     <Image src={rpcs3customconfigchange} alt="RPCS3's right click menu" />
   </div>

2. Switch to the "`Audio`" tab.
   <div>
     <Image src={audiom} alt="Audio settings" />
   </div>

3. ![A tan square with a solid outline.](/milohaxguides/assets/config/smalltan.png "Tan Square") **To set microphones up:**
   - **Set “`Microphone Type`” to “`Standard`”.**
   - **Select an input device in “Mic1”, “Mic2”, and “Mic3”.**
   <div></div>

4. When you're done, click “`Apply`” then “`Save custom configuration`”.
</Steps>
<Aside type="danger">
**Make sure you have [something set up to control your game](https://rb3pc.milohax.org/ctrls#gamepads).** You can't select a song to sing otherwise.
</Aside>
