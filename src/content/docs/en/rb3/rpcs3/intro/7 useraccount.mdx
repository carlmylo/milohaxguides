By default, RPCS3's default username is "`User`" when you're playing offline.  
<Image src={rb3user} alt="Rock Band 3's menu, showing the username set to the default 'User.'" />

Changing it is very easy!
<Steps>
1. In RPCS3, go to **Manage > User Accounts**  
    <Image src={rpcs3user} alt="RPCS3 showing 'User Accounts' option under the 'Manage' menu." />

2. Once you're there, **click on the default username (00000001 - User) and then click "Rename User".**  
    <Image src={rpcs3rename} alt="RPCS3's 'User Manager', showing the default username." />

3. **Now change it** to whatever you want, then close it out.  
    <Image src={rpcs3namepanel} alt="RPCS3's 'Rename User', showing the user changing the name." />
</Steps>
